# Pakistan Student Employability Analysis

Data cleaning, exploratory data analysis, and a documentation report on a dataset of 50,000 Pakistani university students, exploring how academics, skills, and career-readiness activities relate to employability.

## Task

Cleaned a public dataset using Pandas, handled missing values and inconsistent categorical entries, and produced summary statistics and visualizations using Matplotlib and Seaborn, followed by a written findings and recommendations report.

## Key Finding

91.73 percent of students in this dataset are rated Highly Employable, with CGPA, programming skill level, and internship experience showing the clearest links to that outcome.
